# 🌡️ Temperature Calculator (Python)

A simple Python project that converts temperatures between **Celsius and Fahrenheit** using basic formulas. This project is designed for beginners to understand Python fundamentals like input/output, conditional statements, and arithmetic operations.

---

## 🚀 Features

- Convert Fahrenheit to Celsius  
- Convert Celsius to Fahrenheit  
- Simple menu-based system  
- Easy user input handling  
- Rounded output for accuracy (2 decimal places)  
- Beginner-friendly Python logic  

---

## 🧮 Formulas Used

- Celsius = (Fahrenheit - 32) * 5/9  
- Fahrenheit = (Celsius * 9/5) + 32
